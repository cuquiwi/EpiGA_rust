EpiGA\_rust
==========

Contents
---------

This is an implementation of the algorithm exposed in [Epigenetics](https://www.sciencedirect.com/science/article/pii/S0020025517309921) in Rust.

Example how to run
------------------

`cargo run -- --problem problems/a280.tsp --epochsmax 200`
